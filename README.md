# React Login & Registration

A simple React prototype demonstrating login and registration forms with inline validation.

## Getting started

1. Install dependencies
   ```bash
   cd frontend
   npm install
   ```

2. Start development server
   ```bash
   npm start
   ```

3. Build for production
   ```bash
   npm run build
   ```

4. Run tests (optional)
   ```bash
   npm test
   ```

## Notes

- This project is frontend-only; form submission is simulated and validated client-side.
- Switching between login and registration clears the form state and messages.
